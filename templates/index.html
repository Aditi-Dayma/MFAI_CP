PCA — Eigenfaces

Principal Component Analysis reduces facial image dimensionality by projecting faces into a lower-dimensional eigenspace. The method computes eigenvectors (eigenfaces) from the covariance matrix of training images, retaining the most discriminative features for recognition via Euclidean distance matching.

Dimensionality Reduction Linear Algebra Unsupervised

CNN — Deep Learning

Convolutional Neural Networks learn hierarchical spatial features directly from raw pixel data through convolutional filters, pooling layers, and fully connected classifiers. CNNs automatically extract discriminative facial features, offering superior robustness to variations in pose, illumination, and expression.

Deep Learning Feature Extraction Supervised
